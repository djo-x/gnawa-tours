# Project Overview — Gnaoua Tours

## What This Project Is
Gnaoua Tours is a premium travel agency platform focused on Sahara experiences in Algeria.  
It combines a cinematic public website for visitors and a practical admin panel for daily operations.

## Main Goal
Turn visitors into qualified bookings through immersive storytelling, clear program information, and a smooth booking flow.

## Public Website
- Artistic, high-end visual design with smooth motion and section-based navigation.
- Hero experience with configurable background (classic image or optimized Three.js desert scene).
- Program discovery with dates, itinerary, tags, and regional pricing display.
- Dynamic content sections driven by database content.
- Booking form integrated with backend persistence.
- Optional ambient music player with admin-controlled tracks.

## Admin Panel
- Manage programs, dynamic sections, media, and global site settings.
- Upload images/audio to Supabase storage and reuse them across the site.
- Configure logo, hero behavior, contact details, showcase media, and music toggle.
- Booking management with improved dashboard insights and metrics.

## Data & Backend
- Supabase is used for database + storage.
- Schema evolves through SQL migrations (pricing split, music settings, hero mode, logo setting, etc.).
- Seeds are available in English/French variants.

## Current State
This is a polished V1/V2-ready foundation: strong visual identity, admin configurability, production-oriented structure, and room for growth (notifications, automation, CRM-like workflows).
